# Stock the Shelf

## Procedure

1. **Read the source of truth:** `phonebooth/docs/sales/lead-generation-guide.md`. It defines the method, proxies, exclusions, and output format. Do not improvise past it.
2. **Check governing scope:** the most recent `gtm-amendment-YYYY-MM-DD` in Drive. The amendment's segment rulings override the guide's inline scope. (As of 2026-07-07: manufacturers active per DR-2026-07-06-03; contractors paused; warehouses wound down to callbacks.)
3. **Execute the guide.** Corridor-by-corridor Maps queries, keep/skip per the rules, ≤ 2 minutes per lead, 30–45 minutes total.
4. **Deliver:** the tracker-ready table (row 1 = seal-breaker) plus the three-line summary. Stop there.

## Hard rules

- Every phone number traces to a live listing. Numbers are **never** invented, inferred, or recalled from memory — a lead without a listed number is a skip.
- Uncertainty is flagged (`?`), never silently resolved. Sean's 30-second skim is the judgment layer.
- No contact of any kind with any business. List production only.
- No scope creep: no outreach drafts, no company deep-dives, no strategy commentary beyond the three-line summary. If the run surfaces something strategic, note it in one line for Sean to raise with his advisor session.
- If the guide and the latest amendment conflict, the amendment wins — and note the conflict in the summary so the guide gets updated.
